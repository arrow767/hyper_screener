# Примеры конфигурации

## Консервативный (только крупнейшие заявки)

Для отслеживания только самых крупных заявок близко к цене:

```env
MIN_ORDER_SIZE_USD=5000000
MIN_ORDER_SIZE_USD_OVERRIDES=BTC:8000000,ETH:6000000
MAX_DISTANCE_PERCENT=0.1
ALERT_COOLDOWN_MS=300000
LOG_LEVEL=info
```

**Характеристики:**
- Заявки от 5М$
- Расстояние до 0.1% от цены
- Не чаще 1 алерта в 5 минут на заявку
- Минимум шума, только самые важные сигналы

---

## Сбалансированный (по умолчанию)

Хороший баланс между количеством алертов и их важностью:

```env
MIN_ORDER_SIZE_USD=2000000
MIN_ORDER_SIZE_USD_OVERRIDES=BTC:4000000,ETH:3000000
MAX_DISTANCE_PERCENT=0.2
ALERT_COOLDOWN_MS=60000
LOG_LEVEL=info
```

**Характеристики:**
- Заявки от 2М$
- Расстояние до 0.2% (глубина стакана Hyperliquid)
- Не чаще 1 алерта в минуту на заявку
- Оптимально для большинства трейдеров

---

## Агрессивный (все значимые заявки)

Для отслеживания всех потенциально важных заявок:

```env
MIN_ORDER_SIZE_USD=500000
MAX_DISTANCE_PERCENT=0.5
ALERT_COOLDOWN_MS=30000
LOG_LEVEL=info
```

**Характеристики:**
- Заявки от 500К$
- Расстояние до 0.5% от цены
- Не чаще 1 алерта в 30 секунд на заявку
- Больше сигналов, требует фильтрации

---

## Скальпинг / Внутридневная торговля

Для активных трейдеров, следящих за краткосрочными движениями:

```env
MIN_ORDER_SIZE_USD=1000000
MAX_DISTANCE_PERCENT=0.15
ALERT_COOLDOWN_MS=15000
LOG_LEVEL=info
```

**Характеристики:**
- Заявки от 1М$
- Расстояние до 0.15%
- Не чаще 1 алерта в 15 секунд
- Быстрые сигналы для скальпинга

---

## Свинг трейдинг / Позиционная торговля

Для долгосрочных позиций с фокусом на крупные уровни:

```env
MIN_ORDER_SIZE_USD=3000000
MAX_DISTANCE_PERCENT=0.3
ALERT_COOLDOWN_MS=600000
LOG_LEVEL=info
```

**Характеристики:**
- Заявки от 3М$
- Расстояние до 0.3%
- Не чаще 1 алерта в 10 минут
- Фокус на крупных уровнях поддержки/сопротивления

---

## Тестовый / Отладочный

Для тестирования и настройки бота:

```env
MIN_ORDER_SIZE_USD=100000
MAX_DISTANCE_PERCENT=1.0
ALERT_COOLDOWN_MS=5000
LOG_LEVEL=debug
```

**Характеристики:**
- Заявки от 100К$
- Расстояние до 1%
- Не чаще 1 алерта в 5 секунд
- Отладочные логи
- **Не рекомендуется для продакшн - будет много алертов!**

---

## Только топ монеты

Если хотите мониторить только определенные монеты, нужно модифицировать код.

В `src/orderbook-monitor.ts`, метод `start()`:

```typescript
async start(): Promise<void> {
  // ... existing code ...

  // Вместо subscribeToAllAssets используйте:
  const topCoins = ['BTC', 'ETH', 'SOL', 'ARB', 'OP', 'MATIC'];
  
  for (const coin of topCoins) {
    this.hyperliquid.subscribeToOrderBook(coin, (snapshot) => {
      this.processOrderBook(snapshot);
    });
  }

  // ... rest of code ...
}
```

## Рекомендации по настройке

### MIN_ORDER_SIZE_USD

**Как выбрать:**
1. Посмотрите среднюю ликвидность ваших торгуемых монет
2. Установите порог на уровне 2-5% от средней дневной ликвидности топ уровня
3. Для BTC/ETH: 2-5М$ оптимально
4. Для альткоинов: 500К-1М$ оптимально

### MAX_DISTANCE_PERCENT

**Как выбрать:**
1. Глубина стакана Hyperliquid ~0.2%
2. Для заявок в стакане: 0.1-0.2%
3. Для заявок немного дальше: 0.3-0.5%
4. Больше 1% - обычно не имеет смысла (заявки слишком далеко)

### ALERT_COOLDOWN_MS

**Как выбрать:**
1. Зависит от вашего стиля торговли
2. Скальпинг: 10-30 секунд
3. Внутридневная: 30-60 секунд
4. Свинг: 5-10 минут
5. Позиционная: 10-60 минут

**Совет:** Начните с консервативных настроек и постепенно увеличивайте чувствительность.
